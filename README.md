# 🔐 Password Generator

## Overview
Password Generator is a web application that creates secure and customizable passwords. Users can adjust password length, choose character types (uppercase, lowercase, numbers, and symbols), evaluate password strength, and copy generated passwords instantly.

## ✨ Features
- Generate random secure passwords
- Customize password length (6–24 characters)
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special symbols
- Real-time password strength indicator
- One-click copy to clipboard
- Responsive and user-friendly interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How It Works
1. Select the desired password length.
2. Choose the character types to include.
3. Click **Generate Password**.
4. View the generated password and its strength level.
5. Copy the password using the copy icon.
